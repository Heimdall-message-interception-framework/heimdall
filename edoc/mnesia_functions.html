setup_db/0

setup_db() -> ok

Sets up the mnesia database that is needed to store test runs. If the MIL_MNESIA_DIR env variable is set or an mnesia dir is set manually, the database is started on this node. Otherwise we assume that an mnesia node called "master" is already running.
